# bw_share_downloader

Bandwidth sharer download manager helps speed up download by using bandwidth available to other nodes connected on the same local area network or connected through wifi. Here each downloading node also uses top-notch multi-threading to speed up their download.

To run the application :
  1. Run client/client_ui.py on all nodes which may help in download.
  2. Run server/server_ui.py on node that wishes to instantiate download.
  3. Give download-link, name of file to be saved as and extension and press submit.
  4. Progress is displayed on terminals of both client and server sides.
  5. sample download link:http://www.pdf995.com/samples/pdf.pdf
